# 🎬 AI Movie Recommendation System

An AI-powered Movie Recommendation Web Application built using Python, Streamlit, and Machine Learning.

## 🚀 Features

* Movie recommendation system
* TMDB API integration
* Movie posters
* Ratings and release dates
* Mood-based movie suggestions
* Interactive Streamlit UI
* Secure API handling using `.env`

---

## 🛠 Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* TMDB API
* Requests
* Machine Learning

## ▶️ Run Locally

### Clone Repository

```bash
git clone YOUR_GITHUB_REPO_LINK
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Streamlit App

```bash
streamlit run app.py
```

---

## ⚠️ Note

The `similarity.pkl` model file is excluded because of GitHub file size limits.
Run `train_model.py` locally to regenerate the recommendation model.
